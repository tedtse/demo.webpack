Url Loader

依赖
npm install webpack url-loader html-loader --save-dev
html模板引入图片
<img src="../assets/images/logo_123.png" />
配置文件
module: {
        loaders: [
            {
                test: /\.(png|jpg)$/,
                loader: 'url-loader',
                query: {
                    name: 'static/images/[hash].[ext]',
                    limit: 2048
                }
            },
            {
                test: /\.html$/,
                loader: 'html'
            }
        ]
    }
