File Loader

安装
npm install webpack file-loader html-loader --save-dev
html模板引入图片
<img src="../assets/images/logo_123.png" />
配置文件
module: {
        loaders: [
            {
                test: /\.(png|jpg)$/,
                loader: 'file-loader?name=static/images/[hash].[ext]'
            },
            {
                test: /\.html$/,
                loader: 'html'
            }
        ]
    }
